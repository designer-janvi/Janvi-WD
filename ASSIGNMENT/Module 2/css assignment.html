                            ♦ Embeded style sheets allow you to define styles for the whole HTML document in one place.
                            ♦ Embeded style sheets refer to when you embed style sheet information into an HTML document using the style element.

[15] What are the external style sheets?

[16] What are the advantages and disavantages of using external style sheets? 

                            ♦     Advantages:
                            1) The style of a few documents can be controlled from the site by utilizing them.

                            3) To assemble styles in complex circumstances, selector and grouping strategies are utilized.
                            

                            ♦     Disadvantages:
